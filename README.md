Project Overview

This project focuses on predicting chocolate product prices using Machine Learning techniques.
Multiple regression models were trained and compared to achieve optimal performance.

The project includes full data preprocessing, dimensionality reduction, model tuning, and evaluation using Scikit-learn.

Technologies Used

Python
Scikit-learn
Pandas
NumPy
Matplotlib

Machine Learning Approach
Data Preprocessing

Data cleaning and handling missing values
Feature scaling
Train-test split using train_test_split

Dimensionality Reduction

Applied PCA (Principal Component Analysis) to reduce feature dimensionality and improve model efficiency.

Models Implemented

Decision Tree Regressor

Random Forest Regressor

Model Optimization

Used GridSearchCV for hyperparameter tuning

Built a complete Pipeline integrating preprocessing, PCA, and model training

Model Evaluation

Compared model performance using standard regression metrics

Selected the best-performing model based on cross-validation results

Key Features of the Project

End-to-end ML workflow

Feature engineering and dimensionality reduction

Hyperparameter tuning

Model comparison

Clean and modular pipeline implementation

🎯 Goal

The goal of this project was to build an optimized and scalable regression model capable of predicting chocolate prices accurately while demonstrating best practices in Machine Learning workflow design.
